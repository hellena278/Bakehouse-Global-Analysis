This project finds our best bakery markets and treats by tracking total sales across global countries and cities. Built as an end-to-end data engineering and analytics solution, it processes raw bakehouse transaction records through a Medallion Architecture pipeline (Bronze, Silver, Gold) in Databricks. The pipeline cleans the data to show our highest-earning areas, like Australia and Sydney, while highlighting which specific baked goods sell the most.
